# USDT Indexer Test using Subsquid

This is a simple Subsquid indexer created to track USDT `Transfer` events on the Ethereum mainnet.

## Requirements
Squid CLI
node
docker
git

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Run the processor: `node -r dotenv/config lib/main.js` to start indexing the events

you can follow this tutorial:
https://docs.sqd.ai/sdk/how-to-start/squid-from-scratch/
